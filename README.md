      This planner application is designed to help students generate optimized class schedules based on their academic requirements and personal preferences. The process begins by ingesting a CSV file that lists a student’s available classes. Each entry in the CSV is represented using a specific Course type—such as ComputerScience, Biology, Math, History, Engineering, or Business—and stores details including course name, course ID, instructor name, instructor RateMyProfessor score, campus location, credit value, difficulty level, required status, and a WeeklyTimeBlock that defines course days and times.

      Student preferences include preferred daily start and end times, minimum and maximum break times between classes, preferred campus location, required minimum and maximum credits, the student’s major, and any courses the student identifies as struggle courses. Classes that relate directly to the student’s major—such as ComputerEngineering, ChemicalEngineering, DataScienceAndAnalytics, and other discipline‑specific options—are treated as major‑specific during schedule evaluation.

      Once all class data and student preferences are stored, the ScheduleGenerator creates every possible Schedule that can be formed without time conflicts by comparing WeeklyTimeBlock entries. Each Schedule maintains a collection of selected courses, tracks total credits, calculates breaks between classes, and determines daily start and end times to ensure compatibility with the student’s preferences.

      Each generated schedule is then evaluated using a point‑based scoring system. Points are awarded for required courses, major‑specific courses, and higher instructor ratings. Points are reduced for schedules that include multiple struggle courses, fall outside the preferred credit range, start too early or end too late in the day, occur on an undesired campus, or contain breaks that are shorter or longer than the student’s preferred limits.

      After scoring is complete, the system selects the three highest‑scoring schedules and outputs them to the student’s chosen folder. The final result provides the student with three optimized schedule options tailored to both academic goals and personal scheduling preferences.
